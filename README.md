Expense Tracker 💰

A modern Expense Tracker application developed as a portfolio and Java skill showcase project. The application features a responsive frontend interface with dashboard analytics, expense management pages, and Java-based object-oriented implementation modules.

🚀 Features

-Modern Login Page
-Interactive Dashboard UI
-Add Expense Form
-Expense Reports & Analytics
-Profile Management Page
-Responsive Design
-Sidebar Navigation
-Expense Transaction Display
-Java OOP-based Expense Management Logic

🛠️ Technologies Used

-Frontend
-HTML5
-CSS3
-Bootstrap 5
-JavaScript
-Java Implementation
-Core Java
-OOP Concepts
-ArrayList Collections
